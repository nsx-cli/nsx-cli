# Security Policy

## Supported Versions

The current `1.0.x` line is supported for security fixes.

## Reporting a Vulnerability

If you find a security issue, do not open a public issue.

Send a private report with:

- A short description of the issue
- Impacted versions
- Reproduction steps
- Any proof of concept or logs you can share safely

Preferred channel: GitHub Security Advisories for this repository.

## Response Expectations

We will acknowledge reports as soon as practical, triage impact, and coordinate a fix before public disclosure when possible.

# NSX CLI PRODUCT VISION

A NSX CLI deve ser a principal ferramenta para desenvolvimento NestJS Enterprise.

## Objetivos de Produto

- Substituir gradualmente o Nest CLI em cenarios enterprise
- Integrar Prisma de forma nativa
- Integrar ts-morph para automacao segura de codigo
- Gerar APIs completas com padroes corporativos
- Gerar documentacao tecnica automatica
- Gerar testes automatizados
- Gerar Swagger automaticamente
- Possuir modulo de AI integrado ao fluxo de engenharia
- Possuir Marketplace de plugins e templates
- Possuir extensao oficial para VS Code
- Possuir Doctor para diagnostico de arquitetura e qualidade
- Possuir Analyze para inspecao estatica orientada por regras
- Possuir Plugin System extensivel por terceiros

## Norte Arquitetural

- Arquitetura orientada a camadas e responsabilidades explicitas
- Estrategia baseada em Registry em vez de logica condicional extensa
- Alteracoes em codigo sempre seguras, rastreaveis e idempotentes
- Foco em produtividade sem perder governanca tecnica

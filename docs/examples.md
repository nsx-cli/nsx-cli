# Examples

## Generate Module

```bash
nsx generate module billing
```

## Generate Test

```bash
nsx generate test billing
```

## Make API

```bash
nsx make api billing
```

## Prisma CRUD

```bash
nsx prisma crud User
```

## Analyze and Fix

```bash
nsx analyze
nsx fix --dry-run
nsx fix
```

## Graph

```bash
nsx graph --output .nsx/graph-report.md
```

# Command Reference

## generate

`nsx generate <type> <name>`
`nsx generate test <name>`

## make

`nsx make api <name>`
`nsx make resource <name>`

## prisma

`nsx prisma crud <Model>`

## doctor

`nsx doctor [--output <path>]`

## analyze

`nsx analyze [--output <path>]`

## fix

`nsx fix [--dry-run] [--output <path>]`

## graph

`nsx graph [--output <path>]`

## docs

`nsx docs generate [--output <path>]`

## ai

`nsx ai ask <prompt> [--provider <name>] [--model <name>] [--temperature <value>]`
`nsx ai providers`

## marketplace

`nsx marketplace templates list [--catalog-url <url>]`
`nsx marketplace templates search <query> [--catalog-url <url>]`
`nsx marketplace templates install <packId> [--catalog-url <url>]`
`nsx marketplace templates installed`
